Freecell

This was my first project at General Assembly. It is your classic freecell game, and uses JQuery for most of the functionality. It is unique in that it uses DOM traversal to determine gamestate, instead of storing it. It was a great way to practice DOM traversal and recursion functions.

Check it out here.

DBOnyx

DBOnyx is my current project. It is a complete rework of my second project at General Assembly. It provides up to date World of Warcraft auction house listings of all servers. It also provides price histories and email price alerts for users looking for items at specific prices. It is currently handling between 3 and 4 million auctions per hour.

Check it out here.

Farestart

Farestart was a group project created in one week using Ruby on Rails. It utilizes Uber's api and Lyft's documented fare rates to provide cost estimates to users so that they can choose the lowest fare for their trips. The site also uses geocoding and Google Maps search for faster input and searching.

Check it out here.

Mercurium

Mercurium is a turn-based tactical RPG loosely inspired by games such as Final Fantasy Tactics. It uses a hex map instead of the classic grid system.
